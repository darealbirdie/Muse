# Discord Translation Bot

A real-time voice and text translation bot for Discord communities.

## Features
- Live voice translation
- Text translation
- Multiple language support
- Concurrent user sessions
- Custom channel selection

## Commands
- /start - Initialize your translator
- /translate - Start voice translation
- /texttr - Text translation
- /setchannel - Set output channel
- /hide /show - Toggle original text
- /stop - End session
